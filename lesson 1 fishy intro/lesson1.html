Lesson 1: Fishy Intro 🐠

Using HTML, Canvas, and Kontra.js to move a sprite and add images.

🎯 Goal

By the end of this lesson, you’ll move a fish sprite with the arrow keys, switch from a rectangle to an image sprite, and add a background!

Target: 6th graders (no coding experience)

Tech: HTML + Kontra.js

Step 1: HTML + Canvas + Kontra Setup

HTML is the language that builds websites. It uses <tags> to organize what you see.

Tags like <body> and <canvas> mark sections of your page. The canvas is your game’s stage.

In this step, you’ll set up your HTML structure and a blank blue canvas using CSS.

👉 View Step 1 Code

Step 2: Kontra Setup

Add Kontra.js to your project and connect it to your canvas:

<script src="https://unpkg.com/kontra/kontra.min.js"></script>
kontra.init('game');

Now you’re ready to create game objects, sprites, and loops!

👉 View Step 2 Code

Step 3: Create a Rectangle Sprite

Use Kontra’s Sprite({...}) to create a simple player — a rectangle that can later become a fish!

Key ideas:

  • update() runs every frame to update position.
  • render() draws the sprite.
  • fillRect() draws a rectangle using x, y, width, and height.
👉 View Step 3 Code

Step 4: Game Loop

The kontra.GameLoop({...}) keeps your game running, updating and redrawing the screen ~60 times per second.

Call loop.start() to begin the animation.

👉 View Step 4 Code

Step 5: Arrow Key Movement

Now we add initKeys() so Kontra listens for keyboard input.

Use keyPressed('arrowright') or keyPressed('arrowleft') to move the player.

Try changing movement speed from 2 to 10 — what happens?

👉 View Step 5 Code

Step 6: Swap Rectangle with Images

1️⃣ Load Images

Use kontra.load() to load your background and player images before the game starts.

2️⃣ Add a Background Sprite

Create a sprite for your background (like the ocean).

3️⃣ Replace Rectangle with Image Sprite

Now, use your player.png as the sprite image.

4️⃣ Update Render Order

Draw the background first, then the player image on top.

🎉 Your fish should now appear swimming over the ocean background!
